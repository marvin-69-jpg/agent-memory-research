# Chrys Bader

Practitioner 視角最全面的 agent memory 分析者。提出 9 軸設計框架和 10 種 failure modes，262K views 代表業界對「記憶還沒解決」的共鳴。

## Current Understanding

- 核心主張：long-term memory for LLMs remains unsolved — 目前只能做到「幻覺」，週級別就會崩塌
- 提出 **raw vs derived** 根本光譜 — 所有記憶系統都在這個光譜上選位置，兩端都不行
- 提出 **9 軸設計框架**，可以用來 map 任何新 memory 方案的設計決策
- 列出 **10 種 failure modes**，每種都是實戰中最常見的壞法
- 「Infinite context is not the answer」— 成本線性增長 + attention 退化
- **Evaluation paradox**：驗證記憶系統需要 ground truth，但 ground truth 超過任何 context window
