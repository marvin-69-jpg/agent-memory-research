# Philipp Schmid

HuggingFace 的技術領導。提出「2026 is the year of Agent Harnesses」，定義 harness 在 agent framework 之上的定位，以及 context durability 作為新瓶頸。

## Current Understanding

- Harness ≠ agent ≠ framework — 是包裹 model 的 infrastructure，管理 long-running tasks
- 識別出三個 harness 價值：real-world validation、empowering UX、hill climbing via feedback
- 提出 context durability 概念 — harness 是解 model drift 的工具
