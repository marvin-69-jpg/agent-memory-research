# Yohei Nakajima

BabyAGI 創造者。2025 年發表「The Rise of AI Memory」全景分析 — 從消費者產品到 API、開源框架、核心技術、戰略觀點，覆蓋最廣。

## Current Understanding

- 整理了消費者端（ChatGPT / Claude / Gemini / Grok）的記憶做法比較
- 整理了開發者工具生態（Mem0 / Letta / Zep / Cognee / Plastic Labs）
- 提出 **"Memory's Plaid"** 概念 — 一個中立層讓使用者跨 assistant 攜帶記憶，對抗 [[memory-lock-in]]
- 指出大多數系統用 **dual-memory architecture**：fast short-term state + scalable long-term storage
