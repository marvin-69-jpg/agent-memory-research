# Deep Agents

LangChain 推出的開源 agent harness，定位為封閉 harness 的替代方案。

## Current Understanding

- 開源、model agnostic
- 使用 open standards：agents.md、skills
- Memory store plugins：Mongo、Postgres、Redis 等
- 部署方式：(1) LangSmith Deployment（可自架、可帶自己的 database）(2) 任何標準 web hosting
- 由 [[harrison-chase]] 主導，作為「own your memory」論點的實踐
